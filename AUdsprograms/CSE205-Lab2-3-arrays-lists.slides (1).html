def srch(arr,elem):
    for i in range(len(arr)):
        if arr[i] == elem:
            print("found at i")
            return i+1
    return(-1)

Properties of Arrays¶

  • Given an index, you can straightway access the element, for example, $A[3]$
  • Typically (not always!) implemented as contiguous locations in memory for ease of location calculation and hence access.
  • So, given an index $i$, how much time does it take to access $A[i]$?
